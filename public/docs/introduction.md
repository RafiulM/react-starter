# Introduction

Welcome to our comprehensive documentation! This modern React application provides a powerful foundation for building scalable web applications.

## Features

Our application includes:

- **Modern UI Components** - Built with Tailwind CSS and Radix UI
- **Type Safety** - Full TypeScript support throughout the codebase
- **Routing** - TanStack Router for type-safe navigation
- **State Management** - Jotai for atomic state management
- **Authentication** - Better Auth integration
- **Database** - Drizzle ORM with PostgreSQL

## Getting Started

To get started with development:

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`

> **Note:** Make sure you have Node.js 18+ installed on your system.

## Architecture

The application follows a monorepo structure with:

```
/apps
  /app - Main React application
  /api - Backend API
  /web - Astro website
/packages
  /ui - Shared UI components
  /core - Core utilities
```
