# Getting Started

Welcome to the React Starter project! This guide will help you get up and running quickly.

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/rafiulm/react-starter.git
   cd react-starter
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to see your application running.

## Development

The project uses:
- **React 18** with TypeScript
- **Vite** for fast development and building
- **React Router** for client-side routing
- **Styled-components** for CSS-in-JS styling

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run typecheck` - Run TypeScript type checking

## Next Steps

- Customize the components in `src/components/`
- Add new pages in `src/pages/`
- Configure routing in `src/App.tsx`
- Update styles in the theme configuration
